Semiconductor Wafer Defect Classification using different Deep Learning Techniques

Project information

  • Category: Machine Learning, Neural Networks, Deep Learning
  • Project date: November 2022
  • Project URL: www.example.com

  • Deep Learning Techniques: The classification system was built using Deep Learning techniques, including CNNs, CNN with Random Forest, and CNN with Extreme Boost. Convolutional Neural Networks (CNNs) are a type of deep neural network that can extract features from images, making them ideal for image classification tasks. The system leveraged these techniques to analyze images of semiconductor wafers and classify them based on the presence of defects.
  • Image Pre-processing Techniques: The system implemented image pre-processing techniques to enhance the quality of the images before feeding them to the model. The pre-processing techniques included image flipping, rotation, and zoom range. These techniques helped to remove noise from the images and enhance the features, leading to a 10% improvement in accuracy.
  • Defect Classification: The system was designed to classify defects on semiconductor wafers with an average accuracy of 94%. The CNN-based model analyzed the images and identified the presence of any defects in the semiconductor wafers. The accurate classification of defects helped improve the efficiency and cost savings in the semiconductor manufacturing process.
